# Problem: Difference (Codeforces - Assiut Sheet) ⚔️

## 📝 Problem Statement
Given four numbers `A`, `B`, `C` and `D`. 
Print the result of the following equation: 
`X = (A * B) - (C * D)`

## 📥 Input
Only one line containing 4 separated numbers `A`, `B`, `C` and `D`.
**Constraints:** ( -100,000 <= A, B, C, D <= 100,000 )

## 📤 Output
Print `"Difference = "` followed by the result of the equation.

## 🎯 Example 1
**Input:**
1 2 3 4
**Output:**
Difference = -10
*(Explanation: (1 * 2) - (3 * 4) = 2 - 12 = -10)*

## 🎯 Example 2
**Input:**
2 3 4 5
**Output:**
Difference = -14
